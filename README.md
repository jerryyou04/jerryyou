# Jerry You Portfolio

Personal developer portfolio for [jerryyou.com](https://jerryyou.com), built with React and Vite.

## Development

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

The resume is served as a root-level PDF from `public/resume.pdf` so mobile
browsers can use their native PDF viewer. `/resume` and the previous nested URL
redirect to that file in production.
The co-op work reports are standalone static pages in `public/`.
